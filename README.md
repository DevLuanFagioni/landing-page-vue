## Como instalar

1. Clone este repositório:

```bash
git clone https://github.com/
```

2. Acesse o projeto:

```bash
cd landing-page-vue
```

3. Instale as dependências:

```bash
npm install
```

4. Rode o projeto:

```bash
npm run dev
```
